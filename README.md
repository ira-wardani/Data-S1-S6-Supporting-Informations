This repository provides the PBK model equation structures used for nano- and microplastic ingestion and inhalation exposure simulations. 
The repository includes the core ODE systems and initial-condition definitions for transparency and reproducibility.

The full simulation workflow, including uncertainty analysis, quasi-steady-state detection, size-dependent parameterization, deposition-coefficient tables,fitted parameter values, particle-size distribution inputs, calibration datasets, and publication plotting scripts, is not included because these components contain unpublished parameterization and workflow-specific intellectual property.
